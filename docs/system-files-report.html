דוח אחריות קבצים - Boiler Card + Boiler Manager

מסמך זה מתאר מה כל קובץ משמעותי במערכת אחראי לעשות.

מסמך ראשי

קובץאחריות
README.mdתיעוד הפרויקט הראשי: יכולות, התקנה, ארכיטקטורה, שירותים, דוגמאות וקישורים.

אינטגרציית Home Assistant (`custom_components/boiler_manager/`)

קובץאחריות
__init__.pyאתחול האינטגרציה, רישום שירותים, סנכרון קבצי פרונטאנד וניהול resource של Lovelace.
config_flow.pyמסכי ההתקנה והאפשרויות (Config/Options Flow).
const.pyקבועים משותפים: שמות שירותים, מצבים, טוקנים וסכמות.
hebcal_cache.pyשליפה/נירמול נתוני Hebcal וכתיבה לקובץ JSON מקומי לכל entry.
manager.pyמנוע הלוגיקה הראשי: טיימרים, משימות, timeline, חוקים, vacation mode, היסטוריה ושליטה בישות.
manifest.jsonמטא-דאטה של האינטגרציה (domain, version, dependencies, links).
sensor.pyחיישני מצב/מונה ושיקוף חיישנים מהגדרות.
switch.pyישותי Switch דינמיים למשימות.
services.yamlתיעוד סכמות השירותים של האינטגרציה.
strings.jsonמפת מחרוזות בסיסית למסכי אינטגרציה.
translations/en.jsonתרגום אנגלית למסכי האינטגרציה.
translations/he.jsonתרגום עברית למסכי האינטגרציה.
translations/fr.jsonתרגום צרפתית למסכי האינטגרציה.

פרונטאנד מקור (`custom_components/boiler_manager/frontend/`)

קובץאחריות

boiler-card.jsלוגיקת הכרטיס הראשית: render, state sync, אינטראקציות, שירותים, מודאלים.
boiler-shell.jsמעטפת HTML/CSS של הכרטיס וכל המודאלים.
boiler-editor.jsעורך הכרטיס: סכמות שדות, defaults, פרופילים ושינויי קונפיג.
boiler-config.jsברירות מחדל וקבועי קונפיג בצד הפרונטאנד.
boiler-i18n.jsמילון תרגומים לכל הטקסטים בכרטיס ובעורך.
boiler-themes.jsערכות נושא ונירמול theme.
boiler-time-utils.jsפונקציות עזר לעיבוד זמן ומשך.
boiler-timer-paging.jsעימוד גריד הטיימרים במודאל.
boiler-task-export.jsנירמול/בנייה של payload ליצוא משימות.
boiler-task-display.jsפורמוט תצוגת משימות והיסטוריה.
boiler-import-display.jsתצוגת משימות במסך ייבוא.
boiler-history-utils.jsנירמול לוג היסטוריה ויצוא לוג.
boiler-heat-utils.jsחישובי progress/heat וצבעים.
boiler-holiday-rules.jsלוגיקת חגים/שבת ונירמול rules.
boiler-service-utils.jsבדיקת זמינות שירותים וקריאות בטוחות.
boiler-control-services.jsהתאמת שירותי שליטה לפי profile/mode.
boiler-entity-search.jsחיפוש/סינון ישויות לעורך תנאים.
boiler-entity-state-utils.jsנירמול `on/off` לפי קונפיג.
boiler-schedule-days.jsעיבוד ימים ולוקליזציה בלוח תזמון.

קבצי Mirror לפריסה (`www/boiler-card/`)

קובץאחריות
כל קבצי boiler-*.jsמראה (Mirror) של `frontend/` להגשה דרך `/local/boiler-card/`; חייבים להישאר מסונכרנים עם המקור.
README.mdהסבר על תפקיד תיקיית ה־mirror והסנכרון.

חשוב: המקור לפיתוח הוא `custom_components/boiler_manager/frontend/`. תיקיית `www/boiler-card/` היא שכבת mirror לפריסה.
